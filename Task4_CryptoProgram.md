
### Task 4
```
Title: Crypto Program

1. Create a 'Message' Class

    Properties
    ------------------------
    a. ID               -> int
    b. sender           -> String
    c. reciver          -> String
    d. message          -> String
    e. privateMessage   -> String
    e. secretKey        -> int

    Constructor
    ------------------------
    a. ID
    b. Sender

    Behaviours
    ------------------------
    a. get_reciever
    b. get_message
    c. create_SecretKey
    d. encrypt_message
    d. save_message
    e. show_message

2. Behaviour Explanations
   
   a -> Get the reciever of the message from the user

   b -> Get the message text

   c -> Generate a random integer between 0-10 and assign the secretKey value.

   d -> Save the message and all properties as ".txt" file

   e -> Print the message and all properties to the console
   
```
